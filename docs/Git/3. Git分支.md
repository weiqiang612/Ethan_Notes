### 查看分支 git branch

基本语法 git branch

查看分支详细信息 git branch -v 

### 切换分支 git checkout

git checkout -b [分支名] 若未有该分支，先创建该分支，再切换到该分支

git checkout [分支名]

### 合并分支 git merge

git merge [分支名] 将指定分支合并到当前分支上，注意，要先切换到最终分支上再合并 

#### 合并分支时产生冲突：

当当前分支和指定的分支同时对同一个位置进行了不同的修改时，此时合并分支会产生冲突，需要人为决定如何合并

步骤：

1. git status 查看哪些文件有冲突
2. 修改有冲突的文件
3. git add
4. git commit  **注意：不要带文件名提交，两个文件名都一样，git 不知道你要提交哪个文件**
