# TypeScript

### 1.1 介绍

1. TS是微软推出的开源语言
2. TS是JS的超集
3. TS = Type + JS（在JS基础上增加了类型支持）
4. TS文件扩展名为ts
5. TS可编译成标准的JS，并在编译时进行类型检查

### 1.2 使用

1. 全局安装 typescript ：`npm install -g typescript`，安装好后查看ts版本：`tsc -v`
