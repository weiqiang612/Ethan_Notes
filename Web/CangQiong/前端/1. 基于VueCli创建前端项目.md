# 基于vue创建新项目

### 1.1 准备

* node.js 运行环境
* npm 包管理工具
* Vue CLI 脚手架 `npm i @vue/cli -g (vue2)`

### 1.2 创建项目

1. vue create 项目名称
2. vue ui
