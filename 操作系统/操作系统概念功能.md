# 操作系统概念功能

> ### 概念
>
> 1. 负责管理协调硬件、软件等计算机资源的工作
> 2. 为上层用户应用程序提供简单易用的服务
> 3. 是一种系统软件

>### 功能 
>
>#### 一、系统资源的管理者
>
>1.  处理机管理
>
>2.  存储器管理
>3.  文件管理
>4.  设备管理 

>#### 二、 向上层提供方便易用的服务
>
>1.  直接给用户使用的
>   * GUI
>   * 命令接口
>     * 联机命令接口 (cmd)
>     * 脱机命令接口 (.bat)
>2.  给软件、程序员使用的
>    * 程序接口 (即系统调用)
