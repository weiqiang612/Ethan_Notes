# 作业

>1. 作业的组织:作业由程序、数据和作业说明书三部分构成，一个作业可包含多个程序和多个数据集，但至少包含一个程序
>
>2. 作业有哪几个状态，哪个状态进入进程调度的管理范围：
>   * 作业的状态有提交、后备、执行、完成四个状态，执行状态进入进程调度管理范围
>
>3. 作业建立有哪两个过程
>
>   * 作业输入，将作业的程序、数据和作业说明书从输入设备比如键盘磁带磁盘等输入到外存形成初始信息
>
>   * 建立作业控制块JCB，根据作业说明书和作业信息在外存中的位置建立
>
>
>4. Spooling技术的特点有：
>
>   * 实现CPU的充分利用
>
>     * 免除人工干预，硬件成本低
>
>     * 预输入可实现批处理作业预先进入系统形成后备队列，方便作业调度
>
>     * 实现虚设备，通过共享设备来模拟独占型设备的动作
>
